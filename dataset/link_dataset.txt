========================================
DATASET LINKS — CNN vs Transfer Learning
========================================

1. CIFAR-10 (CNN from scratch)
   - Source: University of Toronto / HuggingFace
   - URL: https://www.cs.toronto.edu/~kriz/cifar.html
   - Description: 60,000 32x32 color images, 10 classes
   - We use only 2 classes (cat vs dog: labels 3 and 5)
   - Local: dataset/cifar10/ (pickle files)
   - Docs: https://www.tensorflow.org/api_docs/python/tf/keras/datasets/cifar10

2. Dogs vs Cats (Transfer Learning)
   - Source: Microsoft / Kaggle
   - Download: https://download.microsoft.com/download/3/E/1/3E1C3F21-ECDB-4869-8368-6DEBA77B919F/kagglecatsanddogs_5340.zip
   - Description: 25,000 images of dogs and cats
   - We use a subset of 200 images per class
   - Local: dataset/cats_vs_dogs/ (resized jpg files)

Note: Kedua dataset disimpan di folder dataset/ lokal.
Notebook langsung load dari folder tersebut.
